# Big Robot Skills

Big Robot is a secure AI operations layer for real business workflows. It connects private data, governed actions, custom agents, and human controls so teams can automate work without giving up control.

## Skills

| Skill | What it helps with |
| --- | --- |
| [generate-diagram-from-code](./generate-diagram-from-code/) | Create clear SVG diagrams from code paths, logs, workflows, and technical investigations. |
| [implementation-plan-review](./implementation-plan-review/) | Review, repair, and readiness-check implementation plans before execution with source-backed review lanes and a clear ready / not-ready verdict. |
